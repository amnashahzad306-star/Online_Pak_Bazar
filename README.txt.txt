Online PakBazar Project

Important Instructions

Before running the project, please extract/unzip the complete ZIP/RAR file into a folder on your computer.
If the project is opened directly from the compressed file, images, CSS, and JavaScript files may not load properly.

How to Run the Backend

1. Install Node.js (Version 18 recommended).

2. Install MongoDB.

3. Extract the project folder.

4. Open terminal or command prompt.

5. Navigate to the backend folder.

6. Run the following command:

   npm install

7. Start the backend server using:

   node server.js

8. Make sure MongoDB service is running properly.

How to Run the Frontend

1. Open the extracted frontend folder.
2. Locate the 'index.html' file.
3. Open 'index.html' in Google Chrome or any modern browser.

Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
